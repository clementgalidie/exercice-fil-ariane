# Fiche 02 - créer un fil d'ariane

L'objectif de cet exercice est de créer un fil d'Ariane **uniquement** via *HTML* et *CSS* et compatible jusqu'à **IE 8**.

Principes abordés :
* Accessibilité ;
* Intéropérabilité ;
* Performances.
